CTable Multi-level Header Example

This example demonstrates how to create multi-level (nested) headers using the children property.

// Multi-level header configuration
const tableConfig = {
  Columns: [
    {
      prop: 'user',
      label: 'User Info',
      children: [
        { prop: 'name', label: 'Name', width: 120 },
        { prop: 'age', label: 'Age', width: 80 }
      ]
    },
    {
      prop: 'contact',
      label: 'Contact',
      children: [
        { prop: 'phone', label: 'Phone', width: 150 },
        { prop: 'email', label: 'Email', width: 200 }
      ]
    },
    {
      prop: 'work',
      label: 'Work Info',
      children: [
        { prop: 'department', label: 'Department', width: 120 },
        { prop: 'position', label: 'Position', width: 120 }
      ]
    }
  ],
  headRowStyle: { backgroundColor: '#4a90e2', height: 50 }
};
