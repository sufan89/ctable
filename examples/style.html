CTable Style Examples

This example demonstrates various styling options for CTable including row styles, cell styles, and dynamic styling.

Alternate Row Colors

// Alternate row colors using function
rowStyle: (rowValue, rowIndex) => ({
  height: 35,
  backgroundColor: rowIndex % 2 === 0 ? '#ffffff' : '#f9f9f9'
})

Dynamic Status Colors

// Cell style based on value
cellStyle: (columnInfo, cellValue) => {
  const colors = {
    active: '#4caf50',
    inactive: '#f44336',
    pending: '#ff9800'
  };
  return {
    backgroundColor: colors[cellValue] || '#fff',
    color: colors[cellValue] ? '#fff' : '#333'
  };
}

Custom Header Style

// Custom header styling
headRowStyle: {
  backgroundColor: '#2c3e50',
  height: 45
},
headCellStyle: {
  color: '#ecf0f1',
  fontSize: '15px',
  fontWeight: 'bold',
  fontFamily: 'Arial, sans-serif'
}

Without Borders

// Hide borders
showBorder: false

},
